# ESSENTIA Natuurgeneeskunde

A simple, clean website for ESSENTIA Natuurgeneeskunde practice by Helena Koetje.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment on Vercel

1. Push your code to GitHub
2. Import your repository in Vercel
3. Vercel will automatically detect Vite.js and configure the build settings
4. Your site will be deployed automatically

## Project Structure

```
├── index.html      # Main HTML file
├── style.css       # Stylesheet
├── main.js         # JavaScript file
├── vite.config.js  # Vite configuration
└── package.json    # Dependencies and scripts
```

## Customization

- Edit `style.css` to change colors, fonts, and styling
- Edit `index.html` to modify content
- The header background uses CSS gradients - you can replace with an image by modifying the `.header-background` class
